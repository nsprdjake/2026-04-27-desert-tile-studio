# Desert Tile Studio

A self-contained late-night web toy for generating bold Palm Springs-inspired seamless patterns.

## What it does
- Generates a square repeating tile with six motif families
- Lets you swap between curated desert / neon / lounge palettes
- Exports both the seamless tile and a ready-made portrait poster preview
- Uses a seed so good patterns can be recreated later

## Why it’s fun
It feels like a mini pattern machine for wallpapers, packaging backdrops, merch mockups, social graphics, or just vibey visual play.

## Files
- `index.html` — the full app (HTML, CSS, and JS inline)

Open `index.html` in a browser and start shuffling seeds.
